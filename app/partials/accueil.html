Règles de validations

Audit des codes

Relève les codes unités par unité d'organisation. il y a des listes par:

  • les unités d'organisation sans code
  • les unités d'organisation ayant le meme code
  • les unité d'organisation ayant des codes non conformes
  • les unités d'organisation ayant des codes non harmonisé

Création d'etablissement

Creation d'une unité d'organisation de type etablissement sanitaire. Le code de l'établissement est généré automatique. c'est le code disponible qui est attibué par defaut à l'établissement. Mais un autre code peut être choisi dans la liste disponible

Accès éléments de données

Droit d'acces au éléments de données. gérer les droits acces à certains éléments pour les utilisateurs autre que les superUsers. La particularité c'est d'automatiser l'attribution des droits. l'attribution peut se faire par groupe d'utilisateur bien définis.

Suppression d'instance Tracker

La possibilité de supprimer des instances de tracker.

Etats des formulaires

En selectionnant une unité d'organisation et une période, la liste des formulaires assigné à l'unité d'organisation s'affiche. Les formulaire terminé sont marqué en vert et les autres par une croix rouge. Les formulaire prise en compte sont ceux dont les périodes de rapportage sont:

  • Mensuelle
  • Trimestriel
  • semestrielle
  • Annuelle (de janvier à decembre)

NB: Si la période choisi ne respect pas la périodes de rapportage d'un formulaire ou si notre module ne prend pas en compte le période de rapportage de ce formulaire, un trait bleu indque que ce cas n'est pas prise en compte.

Completude Promtitude

les rapports mensuelle en terme de completude et de promptitude sont affiché pour une unité d'organisation (nation, région sanitaire, district sanitaire, etablissement sanitaire) sélectionné.
NB: Ne sont pas pris en compte les formulaires dont la période de rapportage n'est pas mensuelle
